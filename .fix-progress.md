### Patterns Applied

**Buttons/Pressable:**
```typescript
<Pressable
  accessibilityLabel="Clear description"
  accessibilityRole="button"
  accessibilityHint="What happens"
  accessibilityState={{ disabled: isLoading, selected: isActive }}
>
```

**Headers:**
```typescript
<Text accessibilityRole="header">Title</Text>
```

**Text Inputs:**
```typescript
<TextInput
  accessibilityLabel="Field name"
  accessibilityHint="What to enter"
/>
```

**Sliders/Adjustable:**
```typescript
<Slider
  accessibilityLabel="Current value"
  accessibilityRole="adjustable"
  accessibilityHint="Slide to adjust"
/>
```

**Decorative Elements:**
```typescript
<Icon accessible={false} />
```

**Progress Indicators:**
```typescript
<View
  accessibilityLabel="50% progress"
  accessibilityRole="progressbar"
/>
```
