## Library Selection Criteria for This App

1. **New Architecture support** — Required for RN 0.81+
2. **Expo compatibility** — Must work with Expo managed workflow
3. **TypeScript types** — Required (strict mode)
4. **Active maintenance** — Updated within last 6 months
5. **Privacy-safe** — No telemetry, no data collection
6. **Offline-capable** — Must work without network
