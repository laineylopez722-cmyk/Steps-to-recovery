## Troubleshooting Guide

### Issue: npm install fails with ERESOLVE

**Fix:**

```bash
npm install --legacy-peer-deps
```

### Issue: Expo fails with "Cannot find module 'expo'"

**Fix:**

```bash
cd apps/mobile
npm install expo
```

### Issue: Metro bundler crashes

**Fix:**

```bash
npx expo start --clear
```

### Issue: Android device not detected

**Fix:**

1. Enable USB debugging on device
2. Accept RSA fingerprint prompt
3. Run `adb kill-server && adb start-server`

### Issue: Red screen with "Unable to resolve module"

**Fix:**

1. Stop Metro bundler (Ctrl+C)
2. Clear Metro cache: `npx expo start --clear`
3. If persists, check the specific module path in the error
